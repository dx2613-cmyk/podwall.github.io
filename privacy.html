Политика в отношении обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных составлена в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных и меры по обеспечению их безопасности, принимаемые Обществом с ограниченной ответственностью «Центр Экономических Информационных Систем» (далее — Оператор).

Оператор ставит своей важнейшей целью соблюдение прав и свобод человека и гражданина при обработке его персональных данных, в том числе защиты права на неприкосновенность частной жизни, личную и семейную тайну.

Настоящая политика применяется ко всей информации, которую Оператор может получить о посетителях веб-сайта https://podwall.ru.

2. Основные понятия, используемые в Политике

  • Автоматизированная обработка персональных данных — обработка персональных данных с помощью средств вычислительной техники.
  • Блокирование персональных данных — временное прекращение обработки (за исключением случаев, когда обработка необходима для уточнения данных).
  • Веб-сайт — совокупность графических и информационных материалов, программ и баз данных, доступных по адресу https://podwall.ru.
  • Информационная система персональных данных — совокупность баз данных и технологий, обеспечивающих обработку персональных данных.
  • Обезличивание персональных данных — действия, в результате которых невозможно определить принадлежность данных конкретному лицу без дополнительной информации.
  • Обработка персональных данных — любые действия с персональными данными: сбор, хранение, использование, передача, блокирование, удаление и т.д.
  • Оператор — юридическое лицо, организующее и осуществляющее обработку персональных данных (в данном случае — ООО «ЦЭИС»).
  • Персональные данные — любая информация, относящаяся к прямо или косвенно определённому физическому лицу.
  • Пользователь — любой посетитель сайта https://podwall.ru.
  • Трансграничная передача персональных данных — передача данных на территорию иностранного государства.

3. Персональные данные, которые обрабатывает Оператор

Оператор может обрабатывать следующие персональные данные Пользователя:

  • Фамилия, имя, отчество
  • Электронный адрес (email)
  • Номер телефона
  • Данные, полученные с помощью технологий (cookies, IP-адрес, User-Agent)
  • Обезличенная статистика, собираемая сервисами (например, Яндекс.Метрика, Google Analytics)

Все перечисленные данные далее объединяются понятием «Персональные данные».

4. Цели обработки персональных данных

  1. Обеспечение функционирования сайта и предоставление доступа к его материалам.
  2. Улучшение качества контента и пользовательского опыта.
  3. Проведение анализа посещаемости с использованием сервисов интернет-статистики.
  4. Направление информационных сообщений (рассылок) о новых продуктах, услугах и событиях (только при наличии явного согласия).

5. Правовые основания обработки персональных данных

  1. Обработка осуществляется на основании согласия Пользователя, выраженного путём заполнения форм на сайте.
  2. Обработка обезличенных данных (например, через Яндекс.Метрику) возможна при условии, что пользователь не отключил cookies в настройках браузера.
  3. Обработка может осуществляться в целях исполнения договора (например, при заказе услуги).
  4. Обработка может быть необходима для реализации законных интересов Оператора (в рамках, допустимых по ФЗ-152).

6. Порядок и сроки обработки персональных данных

  1. Персональные данные обрабатываются в течение всего срока функционирования сайта и до достижения целей обработки.
  2. Пользователь в любой момент может отозвать согласие, направив письмо на адрес: ceis.ooo@yandex.ru с пометкой «Отзыв согласия на обработку персональных данных».
  3. После отзыва согласия обработка данных прекращается, а данные уничтожаются или обезличиваются, если иное не требуется по закону.

7. Защита персональных данных

  1. Оператор принимает необходимые правовые, организационные и технические меры для защиты персональных данных от неправомерного или случайного доступа, уничтожения, изменения, блокирования, копирования, распространения.
  2. Доступ к персональным данным имеют только уполномоченные сотрудники Оператора.

8. Передача персональных данных третьим лицам

  1. Персональные данные Пользователя не передаются третьим лицам, за исключением случаев, предусмотренных законодательством РФ (например, по запросу суда, прокуратуры, Роскомнадзора).
  2. Для анализа посещаемости сайт использует сервисы:
    • Яндекс.Метрика
    • Google Analytics
    При использовании Google Analytics применяется функция анонимизации IP-адресов (anonymize_ip), что соответствует требованиям ФЗ-152 и GDPR.

9. Трансграничная передача персональных данных

  1. Оператор осуществляет трансграничную передачу персональных данных при использовании сервисов, размещённых за пределами РФ (в т.ч. Google Analytics, GitHub Pages).
  2. Передача осуществляется с учётом требований ФЗ-152:
    • Пользователь, используя сайт, считается давшим согласие на трансграничную передачу.
    • Сервисы используют стандартные contractual clauses (SCC) и обеспечивают адекватную защиту данных.

10. Права Пользователя

Пользователь вправе:

  • Получать информацию о наличии персональных данных
  • Требовать их уточнения, блокирования или уничтожения
  • Отозвать согласие на обработку
  • Подать жалобу в Роскомнадзор

Для реализации своих прав направляйте запрос на: ceis.ooo@yandex.ru

11. Изменения в политике

  1. Оператор вправе вносить изменения в настоящую политику. Актуальная версия публикуется на сайте по адресу: https://podwall.ru/privacy.html
  2. Политика действует бессрочно до её замены новой редакцией.
